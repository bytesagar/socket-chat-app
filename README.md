# NodeJs & Socket Chat App

### Libraries Used

- Express
- NodeJs
- Socket.io
- bootstrap 

---

### Setup project locally

- `git clone https://github.com/sagar608/socket-chat-app.git`
- cd socket-chat-app
- `npm install`
- `npm run dev`
- The project will start on [localhost:3000](localhost:3000) or [http://127.0.0.1:3000](http://127.0.0.1:3000)
